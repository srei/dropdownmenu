# dropdownmenu

Simple dropdown built with React & React-Redux

Install nodemon: 
```
npm install -g nodemon
```
To run app locally: 
 Install Packages: 
 ```
 npm install 
 ```
 Bundle with Webpack
 ```
 npm run build 
 ```
 Run Server
 ```
 npm start
 ```
 Open Browser To: 
 ``` 
 localhost:9090
 ```
